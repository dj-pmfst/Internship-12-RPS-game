# Rock Paper Scissors Game 

Homework 12 - simple rock paper scissors game

## Features

### Gameplay
- **5-Round Matches**: Each game consists of 5 rounds against the computer
- **Game Review**: Review the choices and results of your previous game
- **Achievements**: Complete achievements to get a reward

### API Integration
- Game data persists to external API (https://api.restful-api.dev/objects)
- Each round is saved with player choice, computer choice, and score

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Styling and animations
- **JavaScript (ES6+)**: Game logic and interactivity
- **Fetch API**: RESTful API integration for data persistence
- **Audio API**: Sound effects and background music


## How to Play

1. **Start a New Game**: Click the "New game" button on the home screen
2. **Begin Playing**: Click "Start" to begin the 5-round match
3. **Make Your Choice**: Click on rock, paper, or scissors
4. **View Results**: See the outcome of each round
5. **Continue**: Click "Continue" to proceed to the next round
6. **Final Results**: After 5 rounds, view your final score and unlock achievements
7. **Review Game**: Click "Review game" to see a breakdown of all rounds

## Installation

- Clone or download this repository
- Open `index.html` in a modern web browser

*Note: Requires JavaScript enabled and internet connection for API functionality*
